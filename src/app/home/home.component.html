Visual sorter website!

Have you ever seen how a sorting algorthm works?

This website shows how each sorting algorithm works.

You can choose the following algoritms:
  • Selection sort
  • Insetion sort
  • Quick sort
  • Bubble sort

System architecture


The ideia is to have the frontend request a backend to sort a list of numbers, and then play a small animation of the sorting algorithm operations.

Each algorithm will play a different animation, due to its design.

Warning:

Calling a sorting algorithm should (most likely) never be done making request to a backend to offload the work.

Nevertheless, the goal in this project is to showcase the communication between a frontend and backend services and what requests are executed when you pressing certain buttons.
