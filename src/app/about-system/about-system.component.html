System architecture


The idea is to have the frontend request a backend to sort a list of numbers, and then play a small animation of the sorting algorithm operations.

Each algorithm will play a different animation, due to its design.

Warning:

In a real system, an HTTP-request to perform a sorting operation should (most likely) never be done. Such an engineering approach is only performed for the purpose of this (small) project. This project goal is to play with HTTP requests and responses. Nevertheless, an argument can be made that we are offloading the work from the endpoint to the server.

In summary, this project goal is to showcase the communication between a frontend and backend services and what requests are executed when you pressing certain buttons.
